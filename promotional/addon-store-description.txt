SHORT DESCRIPTION (one line, ~120 characters)
----------------------------------------------
Interactive screen reader training for NVDA beginners — guided, hands-on drills from inside NVDA itself.


FULL DESCRIPTION
----------------
NVDA Coach is an interactive training add-on built by an assistive technology
instructor for people who are just starting to use NVDA. Instead of reading
commands in a manual, students practice them in real time with spoken
instructions, hints, and progress tracking — all from inside NVDA.

Version 1.0.0 includes 24 lessons across three complete chapters:

  Chapter 1 — Getting Started with NVDA (10 lessons)
    The essential commands every beginner needs: the NVDA key, reading the title
    bar and time, silencing speech, finding your focus, Tab navigation, activating
    controls, reading the current line, Input Help, and the user guide. Several
    lessons open a live accessible practice form alongside the Coach window.

  Chapter 2 — Reading and Moving Through Text (6 lessons)
    Character, word, line, document-boundary, Say All, and text selection
    navigation. Each lesson includes a practice text area embedded directly inside
    the Coach window — no switching to another application required.

  Chapter 3 — Browse Mode and Web Navigation (8 lessons)
    Heading navigation, heading levels, link navigation, form interaction, focus
    mode vs browse mode, landmarks, lists, and the Elements List dialog. A fully
    accessible practice web page opens automatically in the browser.

Key features:
  - Persistent Coach window with spoken, step-by-step instructions
  - Live practice form windows for Tab and control activation lessons
  - Inline practice text areas for reading and navigation drills
  - Practice HTML web page for browse mode lessons
  - Progress tracking saved across NVDA restarts
  - Ctrl+N / Ctrl+B / Ctrl+R to navigate between lessons without the picker
  - Instructor-extensible: lessons are plain, human-readable JSON files
  - Free, no internet connection required after installation

Designed for absolute beginners and anyone who wants structured, systematic
screen reader training they can do independently or with an instructor.
