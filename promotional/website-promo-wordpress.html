No videos, no PDFs, no switching between windows to follow along. Press one key and the Coach begins walking you through what to do and why, one step at a time.

What's New in v1.1.0

  • Sound toggle. Go to NVDA menu → Preferences → Settings → NVDA Coach to turn coaching chimes on or off.
  • Practice windows now open on your cue. A priming step tells you what's coming and gives the Alt+Tab reminder before any practice form appears — no more unexpected focus shifts.
  • Enter key advances after lesson completion. Pressing Enter at the end of any lesson now moves to the next one, consistent with all other key behavior.
  • All prompts spoken aloud. The "Try it now — press Enter to continue" cue is now announced by NVDA automatically, not just shown on screen.
  • NVDA vs. Windows command labels. Every lesson step now tells you whether a command is NVDA-specific or a universal Windows shortcut — important context for transferable skills.
  • Two new chapters covering object navigation and customizing NVDA (keyboard layout, speech rate, voice).
  • Two new browse mode lessons — NVDA Find (NVDA+Ctrl+F) and toggling single-letter navigation (NVDA+Shift+Space).

What Makes It Different

  • Guided, step-by-step lessons. Each step speaks an instruction, tells you which key to press, and waits for you. F1 repeats the instruction, F2 gives a hint, F3 skips the step.
  • Inline practice text. Reading and navigation lessons embed a practice text area directly inside the Coach window. Arrow down from the instruction into real text — no extra windows needed.
  • Live practice forms. Tab navigation and control lessons open a real accessible form alongside the Coach window — actual buttons, checkboxes, and text fields to practice on.
  • Practice web page. Browse mode lessons open a fully accessible HTML practice page in your browser, complete with headings, links, forms, landmarks, and lists.
  • NVDA vs. Windows labels. Every lesson step is labeled so learners understand which commands are NVDA-specific and which work across all screen readers and Windows applications.
  • Progress tracking. Completed lessons are marked in the picker and saved across NVDA restarts so you pick up exactly where you left off.
  • Instructor-extensible. All lessons are plain JSON files. Add a new file to the lessons folder to create custom lesson sets for your students or organization.

35 Lessons Across Five Chapters

Chapter 1: Getting Started with NVDA — 11 lessons

Starts with an orientation to the three categories of keyboard commands — Windows commands, program commands, and screen reader commands — so learners understand who controls what before they touch a single shortcut. Then covers the essential commands every beginner needs: the NVDA modifier key, reading the title bar and time, silencing speech, identifying current focus, Tab navigation, activating buttons and checkboxes, reading the current line, Input Help mode, and opening the user guide. Several lessons include a live accessible practice form alongside the Coach window.

Chapter 2: Reading and Moving Through Text — 6 lessons

Character-by-character navigation, word-by-word navigation, line-by-line navigation, jumping to document start and end, Say All (continuous reading), and text selection with Shift plus arrow keys. Every lesson includes an inline practice text area built directly into the Coach window — no switching to Notepad or another application.

Chapter 3: Browse Mode and Web Navigation — 10 lessons

What browse mode is and how it works, heading navigation, heading level shortcuts, link navigation, form field navigation, toggling between browse mode and focus mode, landmark and list navigation, the Elements List dialog, finding text with NVDA Find, and toggling single-letter navigation. A fully accessible practice web page opens automatically in your browser when you start this chapter.

Chapter 4: Object Navigation — 6 lessons

How NVDA's object tree works, moving to next and previous objects, moving to parent and child objects, reading the current navigator object, routing keyboard focus to any object on screen, and when object navigation is the right tool. Desktop and laptop keyboard layouts are both covered throughout. This chapter unlocks every control on any screen — even those that Tab cannot reach.

Chapter 5: Customizing NVDA — 2 lessons

Changing your keyboard layout between desktop (numpad) and laptop (letter keys), and adjusting speech rate, voice, and synthesizer settings. Includes on-the-fly speed adjustment shortcuts so you can tune NVDA without opening a dialog.

How to Get Started

  1. Make sure NVDA is installed. NVDA is free at nvaccess.org. Any version from 2024.1 onward works.
  2. Download NVDA Coach. Click the download button on this page to get nvdaCoach-1.1.0.nvda-addon.
  3. Open the file. Double-click or press Enter on the downloaded file. NVDA handles the installation automatically and asks you to confirm.
  4. Press NVDA+Shift+C. That is the only shortcut you need to know. The Coach window opens, the lesson picker appears, and you are ready to begin.
  5. Work at your own pace. Each lesson takes 2–5 minutes. Press Ctrl+N after finishing a lesson to move to the next one, or press NVDA+Shift+C at any time to jump to a different chapter.

For AT Instructors and TVIs

NVDA Coach was designed for classroom and one-on-one instruction. Assign a chapter as pre-session homework, use it as a structured warm-up, or give it to students for independent practice between appointments. The lesson picker shows completed lessons at a glance so you and the student can track progress together.
