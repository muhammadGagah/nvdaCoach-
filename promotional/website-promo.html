What Is NVDA Coach?

NVDA Coach is a free add-on for the NVDA screen reader that teaches you commands through short, guided practice sessions — from inside NVDA itself. No videos. No PDFs. No switching between windows to follow along. Press one key and the Coach begins walking you through what to do and why, one step at a time.

"The hardest part of learning a screen reader isn't finding the right resource — it's the gap between reading about a command and actually using one. NVDA Coach closes that gap."

— Tony Gebhard, Assistive Technology Instructor

Built for Real Learning

💡

Command Category Framework

Lesson 1 teaches the three-way framework — Windows commands, program commands, and screen reader commands — so learners understand who controls what before touching a single shortcut.

🎓

Guided, Step-by-Step Lessons

Each step speaks an instruction, tells you which key to press, and waits for you. F1 repeats the instruction, F2 gives a hint, F3 skips the step.

📝

Inline Practice Text

Reading and navigation lessons embed practice text directly in the Coach window. Arrow down from the instruction into real text — no extra windows needed.

📋

Live Practice Forms

Tab navigation and control lessons open a real accessible form alongside the Coach window — actual buttons, checkboxes, and text fields to practice on.

🌐

Practice Web Page

Browse mode lessons open a fully accessible HTML practice page in your browser, with headings, links, forms, landmarks, and lists ready to navigate.

📊

Progress Tracking

Completed lessons are marked in the picker. Your progress saves across NVDA restarts so you can pick up exactly where you left off.

🛠️

Instructor-Extensible

All lessons are plain JSON files. Add a new .json file to create custom lesson sets for your students or organization.

35 Lessons Across Five Chapters

  • Getting Started with NVDA — 11 lessons

    Opens with a dedicated lesson on the three categories of keyboard commands — Windows, program, and screen reader — so learners understand who controls what before any key practice begins. Then: the NVDA key, title bar, time, silencing speech, identifying focus, Tab navigation, activating controls, reading the current line, Input Help, and the user guide.

  • Reading and Moving Through Text — 6 lessons

    Character, word, line, and document-boundary navigation; Say All; and text selection with Shift+arrows. Each lesson includes inline practice text so you can practice without leaving the Coach window.

  • Browse Mode and Web Navigation — 10 lessons

    Heading navigation, heading levels, links, forms, focus mode vs browse mode, landmarks, lists, the Elements List dialog, NVDA Find, and toggling single-letter navigation. A live practice web page opens automatically.

  • Object Navigation — 6 lessons

    The NVDA object tree, moving to next/previous/parent/child objects, reading the current object, and routing keyboard focus to controls that Tab cannot reach. Desktop and laptop layouts documented throughout.

  • Customizing NVDA — 2 lessons

    Switching keyboard layout (desktop vs. laptop) and adjusting speech rate, voice, and synthesizer settings — including on-the-fly speed shortcuts.

How to Get Started

  1. Make sure NVDA is installed.
    NVDA is free at nvaccess.org. Any version 2024.1 or later works.
  2. Download and open the add-on file.
    Download nvdaCoach-1.2.0.nvda-addon using the button above. Open the file — NVDA will install it automatically and ask you to confirm.
  3. Press NVDA+Shift+C.
    That's all. The Coach window opens and you're ready to start. Choose a lesson from the picker, press Enter, and begin.
  4. Work at your own pace.
    Each lesson is self-contained and takes 2–5 minutes. Use Ctrl+N to move to the next lesson at any time.

For AT Instructors and TVIs

NVDA Coach was built by an AT instructor for classroom and one-on-one use. Assign a chapter before a session, use it as a warm-up or structured review, or give it to students to work through independently between appointments. The lesson picker shows progress so you and the student can see at a glance what has been covered.

Need custom lessons for a specific program, organization, or student population? Lessons are plain JSON files — documented and designed to be extended. Get in touch to discuss custom lesson development.
