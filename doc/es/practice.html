Página de Práctica de NVDA Coach

Acerca de esta página

Bienvenido a la página de práctica de NVDA Coach. Esta página fue creada específicamente para practicar la navegación con el lector de pantalla. Contiene encabezados, enlaces, un formulario, una tabla y listas — todo lo que necesitas para practicar las habilidades enseñadas en NVDA Coach.

Puedes navegar esta página en modo exploración usando los atajos de teclado que estás aprendiendo. No es necesario hacer clic en nada. Todo aquí es para practicar, así que siéntete libre de explorar libremente.

Cómo usar esta página

Simplemente presiona las teclas de navegación que te enseña tu lección de NVDA Coach. Por ejemplo, presiona H para saltar entre encabezados, K para saltar entre enlaces y F para saltar entre campos de formulario. Usa la Flecha abajo para leer línea por línea, o Leer todo para leer la página completa (NVDA+Flecha abajo en distribución de escritorio, NVDA+A en distribución de portátil).

Práctica de lectura de texto

Esta sección tiene párrafos más largos para que puedas practicar la lectura y la navegación. NVDA puede leer todo este texto usando Leer todo. En distribución de escritorio presiona NVDA+Flecha abajo; en distribución de portátil presiona NVDA+A. Puedes detener la lectura en cualquier momento presionando Control.

La historia de los lectores de pantalla

Los lectores de pantalla existen desde la década de 1980. Los primeros lectores de pantalla eran programas simples que convertían el texto mostrado en pantalla en voz usando sintetizadores. Con el tiempo, se volvieron mucho más sofisticados, aprendiendo a interpretar gráficos, páginas web y diseños complejos.

Hoy en día, los lectores de pantalla como NVDA, JAWS y VoiceOver son capaces de leer contenido web, hojas de cálculo, correo electrónico y documentos con notable precisión. NVDA, que significa NonVisual Desktop Access, fue lanzado por primera vez en 2006 y se ha convertido en uno de los lectores de pantalla más utilizados en el mundo, en gran parte porque es gratuito y de código abierto.

Por qué importa la práctica

Aprender un lector de pantalla requiere tiempo y repetición. Los atajos de teclado pueden sentirse poco familiares al principio, pero con la práctica se vuelven automáticos. La mayoría de los usuarios experimentados de lectores de pantalla navegan páginas web y documentos a alta velocidad, usando la navegación por encabezados, la navegación por puntos de referencia y las listas de enlaces para encontrar información rápidamente.

La clave es practicar un poco cada día. Concéntrate en un conjunto de comandos a la vez, úsalos en situaciones reales y construye gradualmente tu velocidad y confianza. NVDA Coach está diseñado para ayudarte a hacer exactamente eso.

Práctica de listas

Esta sección tiene una lista desordenada y una lista ordenada. Presiona L para saltar a la siguiente lista. Presiona I para moverte entre elementos dentro de una lista.

Comandos esenciales de NVDA (lista desordenada)

  • NVDA+T — Leer la barra de título de la ventana actual
  • NVDA+Tab — Informar el enfoque actual
  • NVDA+Flecha arriba — Leer la línea actual
  • NVDA+Flecha abajo (escritorio) / NVDA+A (portátil) — Leer todo desde la posición actual
  • NVDA+F12 — Informar la hora (presiona dos veces para la fecha)
  • NVDA+1 — Activar o desactivar el modo de ayuda de entrada
  • NVDA+F7 — Abrir la lista de elementos
  • NVDA+Espacio — Alternar entre modo exploración y modo enfoque
  • Control — Cancelar el habla
  • NVDA+N — Abrir el menú de NVDA

Pasos para navegar una nueva página web (lista ordenada)

  1. Presiona NVDA+T para confirmar en qué página estás.
  2. Presiona H para explorar los encabezados y obtener una visión general de la estructura de la página.
  3. Presiona D para saltar al punto de referencia del contenido principal, saltando la navegación.
  4. Presiona Flecha abajo para leer el contenido de la página.
  5. Usa K para encontrar y seguir los enlaces según sea necesario.
  6. Usa F y E para rellenar cualquier formulario en la página.
  7. Presiona NVDA+F7 para obtener una lista completa de enlaces o encabezados si necesitas encontrar algo específico.
