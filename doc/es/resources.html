NVDA Coach — Entrenamiento adicional y ayuda

Usa los recursos de esta página para continuar tu proceso de aprendizaje de NVDA.

Recursos de NVDA Coach

  • Guía de usuario de NVDA Coach — Documentación completa del complemento NVDA Coach, incluyendo todos los atajos de teclado, descripciones de lecciones y opciones de personalización.
  • Página de práctica de NVDA Coach — La página web usada para las lecciones de modo exploración. Ábrela en cualquier momento para practicar la navegación por encabezados, enlaces, formularios, tablas y listas.
  • NVDA Coach en GitHub — Código fuente, rastreador de problemas, descargas de versiones e historial de versiones.

Recursos oficiales de NVDA

  • NV Access — La organización que crea y mantiene NVDA. Descarga la última versión, obtén soporte y dona para mantener NVDA gratuito.
  • Ayuda y soporte de NVDA — Documentación oficial, foros de la comunidad y opciones de soporte.
  • Guía de usuario de NVDA (en línea) — La referencia completa de NVDA que cubre todas las funciones y comandos.

Recursos de accesibilidad web y lectores de pantalla

  • WebAIM — Guías prácticas y artículos sobre accesibilidad web, incluyendo técnicas de lector de pantalla y navegación por teclado.
  • WebAIM: Usar NVDA para evaluar la accesibilidad web — Una guía práctica para usar NVDA en la navegación web y las pruebas de accesibilidad.
  • Iniciativa de Accesibilidad Web del W3C (WAI) — Estándares y directrices internacionales para el diseño web accesible.
