## Что такое NVDA Coach?

NVDA Coach — это интерактивное обучающее дополнение, которое обучает командам
NVDA с помощью коротких практических упражнений. Вместо того чтобы читать о
командах в руководстве, вы отрабатываете их в реальном времени с речевой
обратной связью. Coach шаг за шагом направляет вас и позволяет двигаться в
вашем собственном темпе, не выходя из NVDA.

NVDA Coach разработан специально для абсолютных новичков и людей, которые
только начинают осваивать скринридеры. Он был создан инструктором по
ассистивным технологиям, который работает с такими учениками каждый день.

## Начало работы

Нажмите **NVDA+Shift+C** , чтобы открыть NVDA Coach. На экране появится
постоянное окно Coach. При первом открытии вы увидите введение и выбор урока.
Используйте клавиши со стрелками для выбора урока и нажмите Enter, чтобы
начать.

Окно Coach остается открытым на протяжении всего занятия, поэтому инструкции
всегда перед вами. Нажмите **NVDA+Shift+C** в любой момент во время урока,
чтобы вернуться к окну Coach, если вы переключились на другое окно для
практики.

## Во время урока

Каждый урок проводит вас через короткую серию шагов. Для шагов, которые просят
вас попробовать команду NVDA, переключитесь на другое окно и выполните
действие, затем нажмите **NVDA+Shift+C** , чтобы вернуться в окно Coach, и
нажмите **Enter** , чтобы подтвердить и продолжить. Для шагов со встроенным
практическим текстом просто перейдите стрелкой вниз в текстовую область внутри
окна Coach и практикуйтесь там напрямую. Следующие клавиши всегда доступны во
время любого урока:

  * **Enter** : Перейти к следующему шагу.
  * **F1** : Повторить текущую инструкцию.
  * **F2** : Получить подсказку для текущего шага. Нажмите снова, чтобы переключаться между тремя подсказками на шаг.
  * **F3** : Пропустить текущий шаг.
  * **Escape** : Остановить текущий урок.
  * **NVDA+Shift+C** : Вернуться в окно Coach из любого другого окна.

## Навигация между уроками

После завершения урока вы можете перейти к следующему, не открывая выбор
заново:

  * **Ctrl+N** : Перейти к следующему уроку в текущей главе.
  * **Ctrl+B** : Вернуться к предыдущему уроку.
  * **Ctrl+R** : Перезапустить текущий урок с начала.
  * **NVDA+Shift+C** : Открыть выбор уроков, чтобы перейти к любому уроку или главе.

## Включенные уроки

### 1\. Начало работы с NVDA

Десять коротких упражнений, охватывающих основные команды, необходимые каждому
новичку. Предыдущий опыт не требуется.

  1. **Познакомьтесь с клавишей NVDA** — Найдите и используйте клавишу-модификатор NVDA (Insert, Caps Lock или Num Pad Zero).
  2. **Прочитайте строку заголовка** — Изучите NVDA+T, чтобы в любой момент определить активное окно.
  3. **Узнайте время** — Изучите NVDA+F12, чтобы услышать текущее время и дату.
  4. **Остановите речь NVDA** — Используйте клавишу Control, чтобы мгновенно отключить речь.
  5. **Узнайте, где вы находитесь** — Изучите NVDA+Tab, чтобы определить элемент с фокусом. Включает живую практическую форму входа.
  6. **Перемещение между элементами с помощью Tab** — Навигация по диалогам и формам с помощью Tab и Shift+Tab. Включает живую практическую форму.
  7. **Активация кнопок и элементов управления** — Используйте Enter и Space для активации того, на чем вы остановились. Включает живые практические элементы управления.
  8. **Прочитайте текущую строку** — Изучите NVDA+Стрелка Вверх, чтобы повторно прочитать любую строку по запросу. Включает встроенный практический текст.
  9. **Изучите клавиши с помощью режима помощи при вводе** — Используйте NVDA+1, чтобы безопасно узнать, что делает любая клавиша.
  10. **Откройте руководство пользователя** — Найдите справочную систему NVDA в меню NVDA.

### 2\. Чтение и перемещение по тексту

Шесть уроков для точной навигации по документам, электронным письмам и
текстовым полям. Каждый урок включает встроенную текстовую область для
практики прямо в окне Coach — не требуется переключаться на другое приложение.

  1. **Перемещение по символам** — Стрелка Вправо и Стрелка Влево для пошагового перемещения по тексту по одному символу.
  2. **Перемещение по словам** — Control+Стрелка Вправо и Control+Стрелка Влево для перемещения по словам.
  3. **Перемещение по строкам** — Стрелка Вниз и Стрелка Вверх для чтения документа строка за строкой.
  4. **Переход к началу и концу** — Control+Home и Control+End для мгновенного перехода к любому концу.
  5. **Чтение всего с текущей позиции** — Say All читает без остановки. Для настольной раскладки: NVDA+Стрелка Вниз. Для ноутбучной: NVDA+A.
  6. **Выделение текста** — Shift + клавиши со стрелками для выделения по символам, словам и строкам; копирование с помощью Control+C.

### 3\. Режим обзора и навигация по веб-страницам

Восемь уроков для навигации по веб-страницам с использованием режима обзора
NVDA. Практическая веб-страница открывается автоматически при запуске любого
урока в этой категории.

  1. **О режиме обзора** — Что такое режим обзора и как переключиться на браузер.
  2. **Навигация по заголовкам** — H и Shift+H для перехода между заголовками на любой странице.
  3. **Переход к уровням заголовков** — Цифровые клавиши от 1 до 6 для перехода к определенным уровням заголовков.
  4. **Навигация по ссылкам** — K и Shift+K для перемещения между ссылками; Tab для интерактивных элементов.
  5. **Взаимодействие с формами** — F, E и B для поиска полей форм, полей ввода и кнопок.
  6. **Режим фокуса и режим обзора** — NVDA+Пробел для переключения между режимами чтения и ввода.
  7. **Ориентиры и списки** — D для перехода между ориентирами страницы; L для перехода между списками.
  8. **Список элементов** — NVDA+F7 для открытия полного списка всех заголовков, ссылок или полей форм.

## Практические среды

Некоторые уроки включают встроенные практические среды, чтобы вы могли
попробовать реальные команды, не выходя из урока:

  * **Живые практические формы** — Уроки по навигации с помощью Tab, активации элементов управления и определению фокуса открывают отдельное доступное окно формы с реальными кнопками, флажками и текстовыми полями для практики.
  * **Встроенный практический текст** — Уроки по чтению и навигации по тексту включают текстовую область для практики прямо внутри окна Coach. Перейдите стрелкой вниз от инструкции к тексту, чтобы попрактиковаться в навигации по символам, словам, строкам и выделению без переключения окон.
  * **Практическая веб-страница** — Уроки по режиму обзора открывают полностью доступную практическую HTML-страницу в вашем браузере с заголовками, ссылками, формами, ориентирами и списками.

## Отслеживание прогресса

NVDA Coach запоминает, какие уроки вы завершили. Завершенные уроки отмечаются
в выборе уроков, чтобы вы могли видеть свой прогресс с первого взгляда. Ваш
прогресс сохраняется в папке конфигурации пользователя NVDA и сохраняется при
перезапуске NVDA.

## Для инструкторов

Уроки хранятся в виде обычных файлов JSON в папке `lessons` дополнения. Каждый
файл JSON определяет одну категорию уроков с упорядоченным списком уроков, а
каждый урок представляет собой список шагов. Шаги могут быть информационными
(ученик читает и нажимает Enter), основанными на жестах (ученик выполняет
действие самостоятельно и подтверждает Enter) или включать встроенный
практический текст для упражнений по навигации в окне. Вы можете создавать
новые категории уроков, добавляя новый файл JSON — используйте прилагаемые
файлы в качестве шаблонов.

## Что нового

### Версия 1.3.1

  * **Завершена инфраструктура локализации:** Добавлены `locale/nvda.pot` (шаблон перевода со всеми 101 переводимой строкой интерфейса), `locale/en/LC_MESSAGES/nvda.po` (английская база) и `TRANSLATING.md` (пошаговое руководство для переводчиков). Эти файлы отсутствовали в версии 1.3 — обёртка `_()` была на месте, но у переводчиков не было файла `.pot` или папки `locale/` для работы. Проблема выявлена Руи Фонтешем из команды португальского перевода NVDA.

### Версия 1.3

  * **Инфраструктура локализации:** Содержимое уроков теперь организовано в языковые подпапки (`lessons/en/`). NVDA Coach автоматически определяет активный язык NVDA и загружает уроки из соответствующей папки (например, `lessons/fr/` для французского), переходя к `lessons/en/` в случае отсутствия. Участники сообщества могут добавлять переведенные наборы уроков, создавая новую языковую подпапку — изменения кода не требуются. Предложено Валентином Куприяновым.
  * **Полная обертка i18n строк:** Все строки интерфейса в коде дополнения теперь обернуты в `_()` для перевода gettext, охватывая надписи кнопок, сообщения о состоянии, речевые объявления, вывод подсказок и весь текст интерфейса в плагине.

### Версия 1.2.2

  * **Исправление ошибки — Кольцо настроек синтезатора:** В уроке ранее утверждалось, что NVDA+Ctrl+Стрелка Вправо/Влево изменяет скорость речи. Это неверно. NVDA+Ctrl+Стрелка Вправо/Влево перемещается между элементами кольца настроек синтезатора (Скорость, Высота тона, Громкость, Голос, Вариант). Скорость речи регулируется с помощью NVDA+Ctrl+Стрелка Вверх (увеличить) и NVDA+Ctrl+Стрелка Вниз (уменьшить). Обе раскладки — настольная и ноутбучная — задокументированы. Сообщено Брэндоном Паттерсоном.
  * **Подсказки по F2 теперь циклические:** Нажатие F2 во время урока циклически переключает до трех подсказок на шаг, объявляя «Подсказка 1 из 3», «Подсказка 2 из 3» и т.д. Индекс подсказки сбрасывается на каждом новом шаге.
  * **Все шаги теперь имеют подсказки:** Каждый шаг во всех пяти главах теперь имеет подсказки, охватывающие расположение клавиш, положение пальцев и реальный контекст. Ранее многие шаги возвращали «нет доступной подсказки».
  * **Полная связность между главами:** Заключительный урок каждой главы теперь называет следующую главу и ссылается на предыдущие уроки, где это уместно.

### Версия 1.2.1

  * **Исправление ошибки — Навигация по объектам, раскладка ноутбука:** В четырех жестах раскладки ноутбука отсутствовала клавиша Shift. Переход к следующему и предыдущему объекту на том же уровне выполняется с помощью NVDA+Shift+Стрелка Вправо и NVDA+Shift+Стрелка Влево. Переход к родительскому объекту и первому дочернему выполняется с помощью NVDA+Shift+Стрелка Вверх и NVDA+Shift+Стрелка Вниз. Без Shift эти сочетания клавиш являются командами курсора рецензирования, которые перемещаются по символам или строкам, а не навигацией по объектам. Сообщено и проверено Джоном Хессом, специалистом по адаптивным технологиям, Государственные службы для слепых.
  * **Исправление ошибки — Урок Say All, раскладка ноутбука:** В уроке Say All ранее отображалось только сочетание клавиш для настольной раскладки (NVDA+Стрелка Вниз). Пользователи ноутбучной раскладки нажимают NVDA+A. Обе раскладки теперь задокументированы в уроке. Сообщено членом сообщества, использующим раскладку ноутбука.
  * **Аудио:** Громкость звука завершения урока увеличена на 15%.

### Версия 1.2.0

  * **Новый урок: Понимание категорий команд** — добавлен как Урок 1 Главы 1. Объясняет трехуровневую структуру (команды Windows, команды программ, команды скринридера) до начала какой-либо практики клавиш. Команды скринридера не выполняют действий — они только информируют.
  * **Полный редизайн звукового сопровождения** — все пять звуковых моментов (начало урока, переход к шагу, подсказка, остановка урока, завершение урока) теперь используют пользовательские файлы WAV вместо сгенерированных сигналов. Переключатель звука в настройках NVDA отключает все звуки.
  * **Справка NVDA Coach в меню Справка NVDA** — Меню NVDA → Справка → Справка NVDA Coach открывает руководство пользователя напрямую.
  * **Клавиши навигации по главам** — Ctrl+N (следующий урок), Ctrl+B (предыдущий урок), Ctrl+R (перезапустить текущий урок).
