NVDA Coach — Тренировочная страница

Об этой странице

Добро пожаловать на тренировочную страницу NVDA Coach. Эта страница создана специально для отработки навигации с помощью экранного диктора. Она содержит заголовки, ссылки, форму, таблицу и списки — всё необходимое для закрепления навыков, которые вы изучаете в NVDA Coach.

Вы можете перемещаться по этой странице в режиме обзора, используя изучаемые сочетания клавиш. Ничего не нужно нажимать мышью. Всё здесь предназначено для тренировки, поэтому смело исследуйте страницу.

Как использовать эту страницу

Просто нажимайте клавиши навигации, которые изучаете в текущем уроке NVDA Coach. Например, нажимайте H для перехода между заголовками, K — между ссылками, F — между полями форм. Используйте стрелку вниз для построчного чтения или NVDA+Стрелка вниз, чтобы прочитать всю страницу целиком.

Практика чтения текста

В этом разделе представлены длинные абзацы, чтобы вы могли отработать чтение и навигацию. NVDA может прочитать весь текст с помощью команды «Прочитать всё», которая запускается сочетанием NVDA плюс стрелка вниз. Вы можете остановить чтение в любой момент, нажав Control.

История экранных дикторов

Экранные дикторы существуют с 1980-х годов. Первые дикторы были простыми программами, которые преобразовывали текст на экране в речь с помощью синтезаторов. Со временем они стали гораздо более сложными, научившись интерпретировать графику, веб-страницы и сложные макеты.

Сегодня экранные дикторы, такие как NVDA, JAWS и VoiceOver, способны читать веб-контент, электронные таблицы, письма и документы с высокой точностью. NVDA (NonVisual Desktop Access) впервые была выпущена в 2006 году и стала одним из самых популярных экранных дикторов в мире, во многом благодаря тому, что она бесплатна и имеет открытый исходный код.

Почему важна практика

Освоение экранного диктора требует времени и повторений. Сочетания клавиш могут казаться непривычными сначала, но с практикой они становятся автоматическими. Большинство опытных пользователей экранных дикторов быстро перемещаются по веб-страницам и документам, используя навигацию по заголовкам, ориентирам и спискам ссылок для быстрого поиска информации.

Ключ к успеху — практиковаться немного каждый день. Сосредоточьтесь на одном наборе команд за раз, используйте их в реальных ситуациях и постепенно наращивайте скорость и уверенность. NVDA Coach создан именно для этого.

Практика работы с таблицами

В этом разделе находится таблица с данными. Нажмите T для перехода к таблице. Оказавшись внутри, используйте Control+Alt+клавиши со стрелками для перемещения между ячейками. NVDA будет объявлять заголовки строк и столбцов по мере перемещения.

Практика работы со списками

В этом разделе представлены как маркированный, так и нумерованный списки. Нажимайте L для перехода к следующему списку. Используйте I для перемещения между элементами внутри списка.

Основные команды NVDA (маркированный список)

  • NVDA+T — прочитать заголовок текущего окна
  • NVDA+Tab — сообщить текущий элемент фокуса
  • NVDA+Стрелка вверх — прочитать текущую строку
  • NVDA+Стрелка вниз — прочитать всё с текущей позиции
  • NVDA+F12 — сообщить время (нажмите дважды для даты)
  • NVDA+1 — включить/выключить режим ввода справки
  • NVDA+F7 — открыть список элементов
  • NVDA+Пробел — переключение между режимом обзора и режимом фокуса
  • Control — остановить речь
  • NVDA+N — открыть меню NVDA

Шаги для навигации по новой веб-странице (нумерованный список)

  1. Нажмите NVDA+T, чтобы убедиться, на какой странице вы находитесь.
  2. Нажмите H для просмотра заголовков и получения общего представления о структуре страницы.
  3. Нажмите D, чтобы перейти к основному ориентиру (main landmark), пропустив навигационную панель.
  4. Используйте стрелку вниз для чтения содержимого страницы.
  5. Используйте K для поиска и перехода по ссылкам по мере необходимости.
  6. Используйте F и E для заполнения форм на странице.
  7. Нажмите NVDA+F7, чтобы получить полный список ссылок или заголовков, если нужно найти что-то конкретное.
