NVDA Coach — Дополнительное обучение и справка

Используйте ресурсы на этой странице, чтобы продолжить изучение NVDA.

Ресурсы NVDA Coach

  • Руководство пользователя NVDA Coach — Полная документация по дополнению NVDA Coach, включая все сочетания клавиш, описания уроков и возможности настройки.
  • Тренировочная страница NVDA Coach — Веб-страница, используемая в уроках по режиму обзора. Открывайте её в любое время, чтобы практиковать навигацию по заголовкам, ссылкам, формам, таблицам и спискам.
  • NVDA Coach на GitHub — Исходный код, отслеживание ошибок, загрузки релизов и история версий.

Официальные ресурсы NVDA

  • NV Access — Организация, которая создаёт и поддерживает NVDA. Загрузите последнюю версию, получите поддержку и сделайте пожертвование, чтобы NVDA оставалась бесплатной.
  • Справка и поддержка NVDA — Официальная документация, форумы сообщества и варианты поддержки.
  • Руководство пользователя NVDA (онлайн) — Полный справочник по NVDA, охватывающий все функции и команды.

Ресурсы по веб-доступности и экранным дикторам

  • WebAIM — Практические руководства и статьи по веб-доступности, включая техники работы с экранными дикторами и клавиатурную навигацию.
  • WebAIM: Использование NVDA для оценки веб-доступности — Практическое руководство по использованию NVDA для навигации в интернете и тестирования доступности.
  • Инициатива W3C по веб-доступности (WAI) — Международные стандарты и рекомендации по созданию доступных веб-сайтов.
