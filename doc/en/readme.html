What is NVDA Coach?

NVDA Coach is an interactive training add-on that teaches you NVDA commands through short, hands-on drills. Instead of reading about commands in a manual, you practice them in real time with spoken feedback. The Coach guides you step by step and keeps you moving at your own pace from inside NVDA itself.

NVDA Coach is designed specifically for absolute beginners and people who are just starting out with screen readers. It was built by an assistive technology instructor who works with these students every day.

Getting Started

Press NVDA+Shift+C to open NVDA Coach. A persistent Coach window will appear on screen. The first time you open it, you will see an introduction and a lesson picker. Use the arrow keys to select a lesson and press Enter to begin.

The Coach window stays open throughout your session so you always have your instructions in front of you. Press NVDA+Shift+C at any time during a lesson to return to the Coach window if you have switched away to practice in another window.

During a Lesson

Each lesson guides you through a short series of steps. For steps that ask you to try an NVDA command, switch to another window and perform the action, then press NVDA+Shift+C to return to the Coach window and press Enter to confirm and continue. For steps with inline practice text, simply arrow down into the text area inside the Coach window and practice there directly. The following keys are always available during any lesson:

  • Enter: Advance to the next step.
  • F1: Repeat the current instruction.
  • F2: Get a hint for the current step. Press again to cycle through up to three hints per step.
  • F3: Skip the current step.
  • Escape: Stop the current lesson.
  • NVDA+Shift+C: Return to the Coach window from any other window.

Navigating Between Lessons

After completing a lesson, you can move to the next one without reopening the picker:

  • Ctrl+N: Move to the next lesson in the current chapter.
  • Ctrl+B: Go back to the previous lesson.
  • Ctrl+R: Restart the current lesson from the beginning.
  • NVDA+Shift+C: Open the lesson picker to jump to any lesson or chapter.

Lessons Included

1. Getting Started with NVDA

Eleven short drills covering the essential commands every beginner needs, beginning with a framework for understanding how NVDA commands work. No prior experience required.

  1. Understanding Command Categories — Learn the three categories of keyboard commands (Windows, program, and screen reader) and why NVDA commands only inform rather than change things.
  2. Meet Your NVDA Key — Find and use the NVDA modifier key (Insert, Caps Lock, or Num Pad Zero).
  3. Read the Title Bar — Learn NVDA+T to identify the active window at any time.
  4. Read the Time — Learn NVDA+F12 to hear the current time and date.
  5. Stop NVDA from Talking — Use the Control key to silence speech instantly.
  6. Find Out Where You Are — Learn NVDA+Tab to identify the focused control. Includes a live practice login form.
  7. Move Between Controls with Tab — Navigate dialogs and forms with Tab and Shift+Tab. Includes a live practice form.
  8. Activate Buttons and Controls — Use Enter and Space to activate what you land on. Includes live practice controls.
  9. Read the Current Line — Learn NVDA+Up Arrow to re-read any line on demand. Includes inline practice text.
  10. Explore Keys with Input Help — Use NVDA+1 to safely discover what any key does.
  11. Open the User Guide — Find the NVDA help system from the NVDA menu.

2. Reading and Moving Through Text

Six lessons for navigating documents, emails, and text fields with precision. Each lesson includes an inline practice text area built directly into the Coach window — no switching to another application required.

  1. Move by Character — Right and Left Arrow to step through text one character at a time.
  2. Jump Word by Word — Control+Right and Control+Left Arrow to move word by word.
  3. Move Line by Line — Down and Up Arrow to read through a document line by line.
  4. Jump to Start and End — Control+Home and Control+End to jump to either end instantly.
  5. Read Everything from Here — Say All reads without stopping. Desktop: NVDA+Down Arrow. Laptop: NVDA+A.
  6. Select and Highlight Text — Shift plus arrow keys to select by character, word, and line; copy with Control+C.

3. Browse Mode and Web Navigation

Ten lessons for navigating web pages using NVDA's browse mode. A practice web page opens automatically when you start any lesson in this category.

  1. About Browse Mode — What browse mode is and how to switch to your browser.
  2. Navigate by Heading — H and Shift+H to jump between headings on any page.
  3. Jump to Heading Levels — Number keys 1 through 6 to jump to specific heading levels.
  4. Navigate by Link — K and Shift+K to move between links; Tab for interactive elements.
  5. Interact with Forms — F, E, and B to find form fields, edit fields, and buttons.
  6. Focus Mode vs Browse Mode — NVDA+Space to toggle between reading and typing modes.
  7. Landmarks and Lists — D to jump between page landmarks; L to jump between lists.
  8. The Elements List — NVDA+F7 to open a full list of all headings, links, or form fields.
  9. Find Text on a Page — NVDA+Ctrl+F to search within a page; NVDA+F3 and NVDA+Shift+F3 to move forward and backward through results.
  10. Toggle Single-Letter Navigation — NVDA+Shift+Space to enable or disable single-letter browse mode navigation keys.

Practice Environments

Several lessons include built-in practice environments so you can try real commands without leaving the lesson:

  • Live practice forms — Lessons on Tab navigation, activating controls, and identifying your focus open a separate accessible form window with real buttons, checkboxes, and text fields to practice on.
  • Inline practice text — Reading and text navigation lessons embed a practice text area directly inside the Coach window. Arrow down from the instruction into the text to practice character, word, line, and selection navigation without switching windows.
  • Practice web page — Browse mode lessons open a fully accessible practice HTML page in your browser with headings, links, forms, landmarks, and lists.

Progress Tracking

NVDA Coach remembers which lessons you have completed. Completed lessons are marked in the lesson picker so you can see your progress at a glance. Your progress is saved in your NVDA user configuration folder and persists across NVDA restarts.

For Instructors

Lessons are stored as plain JSON files in the add-on's lessons folder. Each JSON file defines one lesson category with an ordered list of lessons, and each lesson is a list of steps. Steps can be informational (the student reads and presses Enter), gesture-based (the student performs the action themselves and confirms with Enter), or include inline practice text for in-window navigation drills. You can create new lesson categories by adding a new JSON file — see the included files as templates.

What's New

Version 1.3.2

  • Localization fully activated: addonHandler.initTranslation() was missing from both plugin modules, causing all translated strings to silently fall back to English regardless of the user's NVDA language. Identified by Valentin Kupriyanov.
  • Language-aware doc file resolution: The resources page, browse mode practice page, and user guide now open from the user's language folder (e.g. doc/ru/) when a translation exists, falling back to doc/en/. Previously hardcoded to English regardless of language.
  • Additional strings wrapped for translation: Dialog title, introduction text, welcome screen, and several display text blocks were not accessible to translators. All are now wrapped with _().
  • First complete Russian localization: All UI strings, all five lesson chapters, the user guide, practice page, and resources page are now available in Russian. Contributed by Valentin Kupriyanov, Head of the Russian-speaking NVDA user community (nvda.ru).

Version 1.3.1

  • Localization scaffolding complete: Added locale/nvda.pot (translation template with all 101 translatable UI strings), locale/en/LC_MESSAGES/nvda.po (English base), and TRANSLATING.md (step-by-step contributor guide). These files were missing from v1.3 — the _() wrapping was in place but translators had no .pot file or locale/ folder to work from. Issue identified by Rui Fontes of the NVDA Portuguese translation team.

Version 1.3

  • Localization infrastructure: Lesson content is now organized into language subfolders (lessons/en/). NVDA Coach automatically detects the active NVDA language and loads lessons from the matching folder (e.g. lessons/fr/ for French), falling back to lessons/en/. Community contributors can add translated lesson sets by creating a new language subfolder — no code changes required. Proposed by Valentin Kupriyanov.
  • Full i18n string wrapping: All user-facing strings in the add-on code are now wrapped with _() for gettext translation, covering button labels, status messages, spoken announcements, hint output, and all UI text throughout the plugin.

Version 1.2.2

  • Bug fix — Synth settings ring: The lesson previously stated that NVDA+Ctrl+Right/Left Arrow changes speech speed. This is incorrect. NVDA+Ctrl+Right/Left Arrow navigates between synth settings ring items (Rate, Pitch, Volume, Voice, Variant). Speech speed is adjusted with NVDA+Ctrl+Up Arrow (increase) and NVDA+Ctrl+Down Arrow (decrease). Both desktop and laptop layouts documented throughout. Reported by Brandon Patterson.
  • F2 hints now cycle: Pressing F2 during a lesson cycles through up to three hints per step, announced as "Hint 1 of 3", "Hint 2 of 3", etc. The hint index resets on each new step.
  • All steps fully hinted: Every step across all five chapters now has hints covering key location, finger placement, and real-world context. Previously many steps returned "no hint available."
  • Full cross-chapter connectivity: Every chapter's final lesson now names the next chapter and back-references earlier lessons where relevant.

Version 1.2.1

  • Bug fix — Object Navigation, laptop layout: The Shift key was missing from four laptop layout gestures. Moving to the next and previous object at the same level is NVDA+Shift+Right Arrow and NVDA+Shift+Left Arrow. Moving to the parent and first child is NVDA+Shift+Up Arrow and NVDA+Shift+Down Arrow. Without Shift, those keystrokes are review cursor commands that move by character or line — not object navigation. Reported and verified by John Hess, AT Specialist, State Services for the Blind.
  • Bug fix — Say All lesson, laptop layout: The Say All lesson previously showed only the desktop layout keystroke (NVDA+Down Arrow). Laptop layout users press NVDA+A. Both layouts are now documented throughout the lesson. Reported by a community member using laptop layout.
  • Audio: Lesson complete sound volume raised by 15%.

Version 1.2.0

  • New lesson: Understanding Command Categories — added as Lesson 1 of Chapter 1. Teaches the three-way framework (Windows commands, program commands, screen reader commands) before any key practice begins. Screen reader commands take no action — they only inform.
  • Sound branding overhaul — all five sound moments (lesson start, step advance, hint, lesson stop, lesson complete) now use custom WAV audio files instead of generated beeps. The sound toggle in NVDA Settings disables everything.
  • NVDA Coach Help in the NVDA Help menu — NVDA menu → Help → NVDA Coach Help opens the user guide directly.
  • Chapter navigation keys — Ctrl+N (next lesson), Ctrl+B (previous lesson), Ctrl+R (restart current lesson).
